Detailed Description

template<class ImplClass>
class chip::DeviceLayer::Internal::GenericPlatformManagerImpl_POSIX< ImplClass >

Provides a generic implementation of PlatformManager features that works on any OSAL platform.

This template contains implementations of select features from the PlatformManager abstract interface that are suitable for use on OSAL-based platforms. It is intended to be inherited (directly or indirectly) by the PlatformManagerImpl class, which also appears as the template's ImplClass parameter.
