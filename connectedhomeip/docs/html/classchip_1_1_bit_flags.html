Detailed Description

template<typename FlagsEnum, typename StorageType = typename std::underlying_type_t<FlagsEnum>>
class chip::BitFlags< FlagsEnum, StorageType >

Stores bit flags in a type safe manner.

Template Parameters
FlagsEnumis an enum or (preferably) enum class type.
StorageTypeis the underlying storage type (like uint16_t, uint32_t etc.) and defaults to the underlying storage type of FlagsEnum.
