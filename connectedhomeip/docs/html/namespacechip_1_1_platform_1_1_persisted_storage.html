Read integer value of a key from persistent storage.

Platform is responsible for validating aKey.

Parameters
[in]aKeyA key to a persistently-stored value.
[in,out]aValueA reference to an integer value.
Returns
CHIP_ERROR_INVALID_ARGUMENT if aKey is NULL CHIP_ERROR_INVALID_STRING_LENGTH if aKey exceeds CHIP_CONFIG_PERSISTED_STORAGE_MAX_KEY_LENGTH CHIP_ERROR_PERSISTED_STORAGE_VALUE_NOT_FOUND if aKey does not exist CHIP_NO_ERROR otherwise

CHIP_ERROR chip::Platform::PersistedStorage::Write ( Key  aKey,
uint32_t  aValue 
)

Write the integer value of a key to persistent storage.

Platform is responsible for validating aKey. If aKey does not exist, it will be created and assigned aValue. Otherwise any existing value of aKey will be replaced with aValue.

Parameters
[in]aKeyA key to a persistently-stored value.
[in]aValueThe value.
Returns
CHIP_ERROR_INVALID_ARGUMENT if aKey is NULL CHIP_ERROR_INVALID_STRING_LENGTH if aKey exceeds CHIP_CONFIG_PERSISTED_STORAGE_MAX_KEY_LENGTH CHIP_NO_ERROR otherwise
