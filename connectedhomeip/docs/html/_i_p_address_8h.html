Detailed Description

This file defines the class Inet::IPAddress and related enumerated constants. The CHIP Inet Layer uses objects of this class to represent Internet protocol addresses of both IPv4 and IPv6 address families. (IPv4 addresses are stored internally in the V4COMPAT format, reserved for that purpose.)
