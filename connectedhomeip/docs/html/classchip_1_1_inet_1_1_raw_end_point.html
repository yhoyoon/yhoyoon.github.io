Detailed Description

Objects of this class represent raw IP network endpoints.

CHIP Inet Layer encapsulates methods for interacting with IP network endpoints (SOCK_RAW sockets on Linux and BSD-derived systems) or LwIP raw protocol control blocks, as the system is configured accordingly.

INET_ERROR chip::Inet::RawEndPoint::Listen ( )

Prepare the endpoint to receive ICMP messages.

Return values
INET_NO_ERRORalways returned.

If mState is already kState_Listening, then no operation is performed, otherwise the mState is set to kState_Listening and the endpoint is prepared to received ICMPv6 messages, according to the semantics of the platform.
