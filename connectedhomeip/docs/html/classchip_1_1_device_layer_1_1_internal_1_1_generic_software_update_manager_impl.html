Detailed Description

template<class ImplClass>
class chip::DeviceLayer::Internal::GenericSoftwareUpdateManagerImpl< ImplClass >

Provides a generic implementation of Software Update Manager features that works on multiple platforms.

This template contains implementations of select features from the SoftwareUpdateManager abstract interface that are suitable for use on all platforms. It is intended to be inherited (directly or indirectly) by the SoftwareUpdateManagerImpl class, which also appears as the template's ImplClass parameter.
