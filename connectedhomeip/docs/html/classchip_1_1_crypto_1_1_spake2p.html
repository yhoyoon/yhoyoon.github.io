Detailed Description

The below class implements the draft 01 version of the Spake2+ protocol as defined in https://www.ietf.org/id/draft-bar-cfrg-spake2plus-01.html.

The following describes the protocol flows: 

Commissioner                     Accessory
------------                     ---------

Init
BeginProver
ComputeRoundOne  ------------->
                                 Init
                                 BeginVerifier
                             /-  ComputeRoundOne
                 <-------------  ComputeRoundTwo
ComputeRoundTwo  ------------->
KeyConfirm                       KeyConfirm
GetKeys                          GetKeys

◆ InitImpl()

virtual CHIP_ERROR chip::Crypto::Spake2p::InitImpl ( const Hash_SHA256_stream *  context)
protectedpure virtual

Initialize underlying implementation curve, points, field elements, etc.

The implementation needs to:

  1. Initialize each of the points below and set the relevant pointers on the class: a. M b. N c. G d. X e. Y f. L g. Z h. V

    As an example: this.M = implementation_alloc_point();

  2. Initialize each of the field elements below and set the relevant pointers on the class: a. w0 b. w1 c. xy d. tempbn
  3. The hashing context should be initialized
Returns
Returns a CHIP_ERROR on error, CHIP_NO_ERROR otherwise
