Detailed Description

template<class Writer>
class chip::Encoding::PacketBufferWriterBase< Writer >

BufferWriter backed by packet buffer.

Typical use:

PacketBufferWriter buf(maximumLength);
if (buf.IsNull()) { return CHIP_ERROR_NO_MEMORY; }
buf.Put(...);
...
PacketBufferHandle handle = buf.Finalize();
if (buf.IsNull()) { return CHIP_ERROR_BUFFER_TOO_SMALL; }
// valid data

System::PacketBufferHandle chip::Encoding::PacketBufferWriterBase< Writer >::Finalize ( )
inline

Obtain the backing packet buffer, if it is valid.

If construction succeeded, Finalize() has not already been called, and BufferWriter::Fit() is true, the caller takes ownership of a buffer containing the desired data. Otherwise, the returned handle tests null, and any underlying storage has been released.

Returns
A packet buffer handle.

bool chip::Encoding::PacketBufferWriterBase< Writer >::IsNull ( ) const
inline

Test whether this PacketBufferWriter is null, or conversely owns a PacketBuffer.

Return values
trueThe PacketBufferWriter is null; it does not own a PacketBuffer. This implies either that construction failed, or that Finalize() has previously been called to release the buffer.
falseThe PacketBufferWriter owns a PacketBuffer, which can be written using BufferWriter Put() methods, and (assuming no overflow) obtained by calling Finalize().
