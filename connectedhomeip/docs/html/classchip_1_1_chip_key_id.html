Detailed Description

The definition of the CHIP Key identifier. This class contains key types, key flags, key ID fields definition, and API functions.

Public CHIP key ID fields, flags, and types.

Enumerator
kType_None 

CHIP key types used for CHIP message encryption.

Note
16 (out of 32) most significant bits of the message encryption key type should be zero because only 16 least significant bits of the ID are encoded in the CHIP message. CHIP message is unencrypted.
kType_General 

General key type.

kType_Session 

Session key type.

kType_AppStaticKey 

Application static key type.

kType_AppRotatingKey 

Application rotating key type.

kType_AppRootKey 

CHIP key types (other than CHIP message encryption types).

Note
16 (out of 32) most significant bits of these types cannot be all zeros, because these values are reserved for the CHIP message encryption keys only.

Constituent group key types. Application group root key type.
