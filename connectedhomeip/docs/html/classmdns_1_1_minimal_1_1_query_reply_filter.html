QueryReplyFilter& mdns::Minimal::QueryReplyFilter::SetIgnoreNameMatch ( bool  ignore)
inline

Ignore qname matches during Accept calls (if set to true, only qtype and qclass are matched).

Ignoring qname is useful when sending related data replies: cliens often query for PTR records however to avoid chattyness, servers generally send all of PTR, SRV (pointed to by PTR) and A/AAAA (pointed to by SRV).
