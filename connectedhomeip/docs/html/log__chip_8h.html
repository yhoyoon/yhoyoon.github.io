Redirect pigweed log prints to chip logging.

Currently does not use a log module, but this could be added when needed. pigweed debug and info level logs are mapped to chip's kLogCategory_Detail. All other log levels are mapped to chip's kLogCategory_Error.

Note: This function should not be called directly, instead call the functions in pw_log/log.h.
