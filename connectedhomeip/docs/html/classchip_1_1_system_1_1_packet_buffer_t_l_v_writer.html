CHIP_ERROR chip::System::PacketBufferTLVWriter::Finalize ( chip::System::PacketBufferHandle *  outBuffer)
inline

Finish the writing of a TLV encoding and release ownership of the underlying PacketBuffer.

Parameters
[in,out]outBufferThe backing packet buffer.
Return values
CHIP_NO_ERRORIf the encoding was finalized successfully.
CHIP_ERROR_TLV_CONTAINER_OPENIf a container writer has been opened on the current writer and not yet closed.
CHIP_ERROR_INVALID_ARGUMENTIf the apparent data length does not fit in uint16_t.
Note
No further TLV operations may be performed, unless or until this PacketBufferTLVWriter is re-initialized.
