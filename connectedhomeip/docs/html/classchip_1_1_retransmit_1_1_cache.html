Detailed Description

template<typename KeyType, typename PayloadType, size_t N>
class chip::Retransmit::Cache< KeyType, PayloadType, N >

This class maintains a cache of data that is sufficient to retransmit.

Typical use is to keep track of unacknowledged packets and resend them as needed.

Template Parameters
KeyTypethe key to identify a single message
PayloadTypethe type of payload to cache for the given peer address
Nsize of the available cache

PayloadType MUST provide a way to reference count, as the data will preserved until the cache is freed:

PayloadType chip::Retransmit::Aquire(PayloadType&); chip::Retransmit::Release(PayloadType&);
