Detailed Description

A class for managing a counter as an integer value intended to persist across reboots.

Counter values are always set to start at a multiple of a bootup value "epoch".

Example:

  • Assuming epoch is 100 via LifetimePersistedCounter::Init(_, 100) and GetValue + AdvanceValue is called, we get the following outputs:
    • Output: 0, 1, 2, 3, 4
